# 介绍
用Django框架试一下搭建一个个人博客。

#版本
Django:  1.9.1
Python:  2.7

# 构建步骤
1. 首先下载源代码

2. 在Mysql中创建数据库，数据库名mydb

3. 修改Myblog/setting.py 中的配置

4. 在当前目录下打开命令行，运行
python manage.py migrate
创建数据库
